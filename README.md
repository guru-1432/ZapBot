## Steps for setting up the Flask app

-  Web scraping to find the class_id should be done prior to start of the application.
-  slack webhook to be include in the config file for get notification.
-  update the web url for the prooduct to be scraped.

### Note
This application is to automate the process of checking the price manualy and get notification.

## prerequisite:
- Python 3.5
- Install all the requirement from requirement.txt using the below code (Creating a virtual environment is recomended)
- ` pip intall -r requirement.txt`
